# Square Project ⬜

A modular web application built with multiple JavaScript components.

## Features

- **Modular Architecture** - JavaScript split into 5 organized parts
- **Complete Web Application** - Full HTML/CSS/JS implementation
- **Scalable Structure** - Easy to maintain and extend
- **Component-Based Design** - Separated functionality

## File Structure

```
sqaure/
├── index.html          # Main HTML file
├── style.css           # Styling and layout
├── script-part1.js     # First component/module
├── script-part2.js     # Second component/module
├── script-part3.js     # Third component/module
├── script-part4.js     # Fourth component/module
└── script-part5.js     # Fifth component/module
```

## Architecture

The project is organized into modular JavaScript files:
- **script-part1.js** - Core functionality and initialization
- **script-part2.js** - User interface components
- **script-part3.js** - Data handling and processing
- **script-part4.js** - Animation and visual effects
- **script-part5.js** - Utility functions and helpers

## Getting Started

1. Clone the repository
2. Open `index.html` in a web browser
3. The application will load all modules automatically

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling
- **Vanilla JavaScript** - ES6+ features
- **Modular Design** - Separated concerns

## Development

The modular structure allows for:
- **Easy Maintenance** - Each module has a specific purpose
- **Team Collaboration** - Multiple developers can work on different parts
- **Code Reusability** - Modules can be reused in other projects
- **Testing** - Individual modules can be tested separately

## Browser Compatibility

- **Chrome** - Full support

- **Safari** - Full support
- **Edge** - Full support
